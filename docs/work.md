该项目的基本流程如下：

1. 在微信小程序端默认写死几个联邦学习任务的分类

2. 完成分类列表的返回接口

3. {后台}，能够在分类管理中查看到分类信息

4. 创建任务

输入任务名称，任务封面，任务简介（富文本框）

输入模型的超参数 数据格式 模型文件地址 

联邦学习任务的训练次数，每个客户端

5. 查看任务列表
