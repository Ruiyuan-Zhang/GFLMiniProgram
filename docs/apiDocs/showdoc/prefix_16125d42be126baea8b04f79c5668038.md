

    
##### 简要描述

- 用户登录接口

##### 请求URL
- ` http://xx.com/api/user/login `
  
##### 请求方式
- POST 

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|username |是  |string |用户名   |
|password |是  |string | 密码    |


##### 返回示例 

``` 
  {
    "error_code": 0,
    "data": {
      "token": "" ,
    }
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|token|string|token|
